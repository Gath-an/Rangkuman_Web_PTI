Rangkuman Materi CSS

Pengertian CSS


CSS, singkatan dari Cascading Style Sheets, adalah bahasa yang umumnya digunakan untuk mengatur tampilan elemen yang ditulis dalam bahasa markup seperti HTML. Fungsi utama CSS adalah memisahkan konten dari tampilan visual pada sebuah situs web.


CSS pertama kali diciptakan dan dikembangkan oleh W3C (World Wide Web Consortium) pada tahun 1996 untuk alasan yang sederhana. Pada awalnya, HTML tidak memiliki tag yang berfungsi untuk memformat halaman. Pengguna hanya perlu menulis markup untuk situs web.


Tag seperti <font> diperkenalkan dalam HTML versi 3.2, tetapi hal ini menyebabkan banyak masalah bagi pengembang. Karena situs web memiliki berbagai font, warna latar belakang, dan gaya, menulis ulang kode menjadi proses yang panjang dan rumit. Oleh karena itu, W3C menciptakan CSS untuk mengatasi masalah ini.


HTML dan CSS memiliki keterkaitan yang erat. HTML berfungsi sebagai bahasa markup (fondasi situs), sementara CSS memperbaiki gaya (untuk semua aspek terkait tampilan website). Oleh karena itu, kedua bahasa ini harus bekerja sama. Selain penting dari segi teknis, CSS juga mempengaruhi tampilan visual sebuah situs web.


Kelebihan CSS

Cukup mudah membedakan mana halaman website yang menggunakan CSS atau tidak. Anda pasti pernah membuka website, tapi sesaat kemudian gagal loading dan malah menampilkan background putih yang didominasi teks hitam dan biru. Situasi ini menggambarkan bahwa CSS pada situs tersebut tidak dapat di-load dengan benar atau situs sama sekali tidak memiliki CSS. Umumnya, situs seperti ini hanya menerapkan HTML.

Sebelum menggunakan CSS, semua stylizing harus disertakan ke dalam markup HTML. Itu berati Anda harus mendeskripsikan semua background, warna font, alignment, dan lain-lain secara terpisah.

Dengan CSS, Anda dapat mengatur tampilan semua aspek pada file yang berbeda, lalu menentukan style, kemudian mengintegrasikan file CSS di atas markup HTML. Alhasil, markup HTML bisa lebih mudah di-maintain.

Singkatnya, dengan CSS, Anda tidak perlu mendeskripsikan tampilan dari masing-masing elemen secara berulang-ulang. Anda tidak membuang-buang waktu, kode yang digunakan pun lebih singkat, dan error dapat diminimalisir.

Karena opsi kustomisasi yang ada hampir tak terbatas, CSS memungkinkan Anda untuk menerapkan berbagai macam style pada satu halaman HTML.

Cara Kerja CSS


CSS menggunakan bahasa Inggris sederhana dengan sintaks berbasis aturan. Sebagaimana telah dijelaskan sebelumnya, HTML tidak didesain untuk menerapkan elemen gaya, melainkan hanya untuk markup halaman. HTML hanya berfungsi untuk mendeskripsikan konten. Sebagai contoh: <p>This is a paragraph.</p>.


Sintaks CSS cukup mudah dipahami. Struktur ini terdiri dari selektor dan blok deklarasi. Anda memilih elemen yang ingin diubah tampilannya, lalu mendeklarasikan perubahan yang ingin diterapkan pada elemen tersebut.


Terdapat berbagai aturan yang perlu diingat. Namun, struktur aturannya sangat simpel dan mudah dipahami.


Selektor merujuk pada elemen HTML yang ingin Anda ubah tampilannya. Blok deklarasi berisi satu atau lebih deklarasi yang dipisahkan oleh tanda titik koma.


Setiap deklarasi terdiri dari nama properti dan nilainya, yang dipisahkan oleh tanda titik dua. Biasanya, deklarasi CSS diakhiri dengan tanda titik koma, sedangkan blok deklarasi dikelilingi oleh tanda kurung kurawal.



Berikut contoh yang bisa kita lihat:


  • <style>
  • p {
  • text-align: center;
  • font-size: 16px;
  • color: pink;
  • }
  • <style>
